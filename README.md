📌 Overview

ProjectVault is an AI-driven web platform designed to help students generate unique final-year project ideas,
complete with:

📊 Novelty scoring

🧠 AI-based idea generation

💻 Code skeleton generation

🔍 Originality checking

📈 Industry insights

🤖 Interactive AI chatbot

It eliminates the struggle of finding innovative project ideas and provides a complete roadmap from    idea → implementation.

✨ Key Features

🧠 AI Project Idea Generator
Generates fully structured project ideas
Includes:

 1.Problem statement
    
 2.Solution approach
 
 3.Tech stack

 4.Roadmap

 5.Industry alignment
 
 6.Provides novelty score (%)

💻 AI Code Generator

Generates starter code + project structure
Supports:

Python (Flask, FastAPI, Django)

JavaScript (Node.js, React)

Java, C++, Go, etc.

Includes:
Boilerplate code

Folder structure

Config setup

🔍 Originality Checker
Estimates project uniqueness

Detects similar ideas

Provides:

Improvement suggestions

📊 Market Insights Dashboard

Required skills
